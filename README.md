# 结算单转内控结算表（Web）

上传客户结算表与员工工号关联表，生成公司内部内控结算表（Excel 下载）。

## 环境要求

- Python 3.10+（本项目本地测试使用 venv）

## 本地启动

在项目根目录执行：

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
uvicorn app.main:app --reload --port 8000
```

浏览器访问 `http://127.0.0.1:8000/`。

## 使用方式

1. 上传 **客户结算表**（一个 Excel，包含结算与差旅两个 sheet；sheet 名包含“结算单/结算”“差旅明细/差旅”字样）。\n+2. 上传 **员工工号关联表**（两列：`工号`、`姓名`）。\n+3. 点击“生成并下载”。\n+\n+## 关键规则（第一版）\n+\n+- **员工工号**：用关联表 `姓名 -> 工号` 匹配；同名多工号会报错。\n+- **月份解析**：\n+  - 支持 `月度/月份` 为 `4月` / `2025-04` / `2025/04` / `2025年4月`。\n+  - 如果单元格只有 `4月` 这种无年份格式，则要求客户结算表文件名包含年份（如 `2025`）。\n+- **应出勤天数**：内置中国法定节假日与调休补班规则，按“该年月工作日天数”计算。\n+- **结算日单价(不含税)**：`月结算价 / 本月应出勤天数 / 1.06`。\n+- **工时费(不含税)**：`结算工时(天) * 结算日单价(不含税)`。\n+- **差旅费/差旅补贴(不含税)**：差旅明细按“出差人+年月”汇总后分别 `sum(报销合计)/1.06`、`sum(补贴金额)/1.06`。\n+\n+## 常见报错\n+\n+- **找不到 sheet / 多个 sheet 命中**：请检查 sheet 名是否包含关键字且唯一。\n+- **缺少列**：请确认表头与需求提供的格式一致。\n+- **同一人同月多行结算**：第一版会报错（避免取值歧义）。\n+- **文件名缺年份**：当 `月度/月份` 只有 `4月` 时需要文件名带 `2025` 这样的年份。\n+
